1研 研 研究 究 究背 背 背景 景 景與 與 與問 問 問題 題 題描 描 描述 述 述
本研究的核心目標，是透過空間液晶調製器（Spatial Light Modulator, SLM）調製射入波的相
位，使反射波在CCD上形成接近平頂（flat-top）的光強分布。理想情況下，平頂光束應具有均
勻、平坦且邊界可控的強度輪廓，然而在實際實驗中，常受到失焦、散光、球差、像差耦合、光
學元件裝配誤差以及環境擾動等非理想因素影響，使得最終量測到的光強分布存在起伏與不平
整。
目前系統雖已能產生接近平頂的分布，但不平整度難以低於1%，顯示僅依靠人工調整相位參數已
接近極限。尤其當補償模型使用Zernike polynomials描述非理想波前時，若需要調整的參數數量
達15維，便會面臨高維搜尋空間過大、參數彼此耦合以及人工判讀效率過低的問題。因此，本研
究希望發展一套結合深度學習與閉迴路控制的自適應調整方法，讓系統能夠由單張CCD光強圖自
動辨識其所對應的Zernike畸變成分，進一步更新補償參數並反覆迭代，最終突破現有人工調整的
效能上限。
2研 研 研究 究 究動 動 動機 機 機
Zernike多項式是描述圓孔徑波前誤差的標準工具，在光學工程中廣泛用於表
示defocus、astigmatism、coma、spherical aberration等常見像差。由於不同像差模式會在焦平
面或成像面上留下特定的強度分布特徵，因此若能訓練一個模型，讓它直接從CCD量測到的強度
圖回推15維Zernike係數，即可將原本高度依賴經驗的人工調整，轉化為資料驅動的快速辨識與自
動補償流程。
深度學習特別適合處理此類「影像輸入、連續參數輸出」的問題。相較於手工萃取特徵或逐一掃
描參數，卷積神經網路（CNN）可以自動學習光斑紋理、局部對稱性、邊界變形與能量分布等特
徵，並將它們映射到對應的Zernike係數空間。因此，本研究不將問題視為分類，而是定義為多 多 多輸 輸 輸
出 出 出回 回 回歸 歸 歸：輸入一張灰階CCD強度圖，輸出一個15維連續向量
ˆz= (ˆz1,ˆz2, . . . ,ˆz15),
其中ˆzi 表示第i個Zernike mode的估計係數。藉此，模型不僅能做一次性的誤差診斷，也可作為

4.2訓 訓 訓練 練 練資 資 資料 料 料的 的 的模 模 模擬 擬 擬生 生 生成 成 成
研究第一步是建立資料集。由於實驗初期通常缺乏大量帶標籤的真實量測資料，因此將先以模擬
方式生成訓練集。其基本流程如下：
1.定義15個欲補償的Zernike modes與其係數範圍。
2.在每次生成樣本時，從各係數範圍中隨機取樣，形成一組15維參數向量。
3.將各mode線性組合成總波前誤差，疊加於理想平頂光束相位上。
4.經由前向模型計算對應的CCD強度圖，並加入合理的噪聲、位移、增益偏差與解析度限
制，以模擬實際量測情況。
2
5.以生成之強度圖作為輸入，以對應的15維Zernike係數作為標籤，建立監督式學習資料集。
值得注意的是，訓練資料不應只侷限於單一mode單獨擾動，而應以15維隨機疊加的方式生成。
原因在於真實實驗中的像差通常同時存在，且彼此效應會交互耦合。若模型只看過單一mode的樣
本，將難以應對實際環境中的混合畸變。
4.3 CNN回 回 回歸 歸 歸模 模 模型 型 型架 架 架構 構 構
本研究採用CNN作為基礎模型，理由是光強圖中的局部紋理、邊緣變形與空間分布特徵，都適合
透過卷積層進行抽取。模型的輸入與輸出形式如下：
•輸入：1×H×W的單通道灰階CCD強度圖。
•輸出：15維連續值向量，對應15個Zernike係數。
一個可直接執行的demo架構如下：
1. 4層卷積模組，逐層抽取低階到高階空間特徵。
2.每層搭配非線性活化函數與池化操作，以提升模型對局部特徵與尺度變化的描述能力。
3.經過flatten後接上數層全連接層，將影像特徵映射至15維係數空間。
4.輸出層不使用分類機率，而直接輸出15個連續值。
在損失函數方面，選擇SmoothL1Loss作為初始設定。相較於單純的MSE，SmoothL1對少數大誤
差樣本較為穩定，可降低訓練初期因個別難例造成的不穩定振盪。未來若要進一步強化模型對最
終平頂品質的敏感度，也可設計混合損失函數，例如：
L=λ 1Lcoeff +λ 2Lflatness,
其中Lcoeff 為係數預測誤差，L flatness 則根據重建或補償後的光強均勻度定義，使模型不僅學會回
推係數，也朝向降低平頂不平整度的最終應用目標。

5.4 Surrogate forward model的 的 的設 設 設計 計 計邏 邏 邏輯 輯 輯
FlatTopZernikeSimulator是此demo的核心。它的流程不是直接計算完整繞射傳播，而是以
「具光學意義的近似失真機制」來模擬實驗中可能觀察到的強度變形。其步驟如下：
1.先建立理想平頂光斑作為target image。
2.將輸入的15維Zernike係數與基底做線性組合，形成總波前誤差W(x, y)。
3.對波前取空間梯度，得到影像空間中的位移場dx, dy，模擬像差引起的幾何扭曲。
4.依據係數RMS大小決定高斯模糊強度，對應成像失焦或高階畸變造成的模糊。
5.使用波前的Laplacian建立shading項，反映局部能量分布與曲率造成的亮度調變。
6.最後加入隨機雜訊，使資料更接近真實CCD量測。
這種surrogate設計雖然不是完整物理模型，但它可以保留「不同Zernike係數造成不同空間紋理變
化」這項重要特性，因此足以支撐CNN對像差成分的辨識學習。

5.6 CNN回 回 回歸 歸 歸模 模 模型 型 型細 細 細節 節 節
ZernikeRegressor採用四層卷積特徵抽取架構，並在最後接上全連接回歸頭。其設計理念如下：
1.第一至第四層卷積模組逐步將通道數從1提升至128，以擷取從局部亮度變化到高階空間結構
的特徵。
2.每個ConvBlock包含兩層卷積、Batch Normalization與ReLU，可提升特徵表達與訓練穩定
性。
3.中間穿插MaxPool2d進行降採樣，使模型學會較大尺度的空間模式。
4.最後使用AdaptiveAvgPool2d((1,1))壓縮空間維度，接著以全連接層輸出15個連續值。
5.在回歸head中加入少量Dropout，可降低過擬合風險。
此模型屬於中小型CNN，優點是結構清楚、參數量適中、容易訓練，也方便未來依需求升級為更
深的backbone，例如ResNet類型結構。
5.7訓 訓 訓練 練 練流 流 流程 程 程與 與 與損 損 損失 失 失函 函 函數 數 數
程式中的訓練函式train model()採用以下設定：
•損失函數：SmoothL1Loss
•最佳化器：Adam
•學習率排程：CosineAnnealingLR
•訓練epoch：demo預設為5

在每一個epoch中，程式都會同時計算訓練集與驗證集上的loss與mean absolute error。這種設計

由於此問題本質上是連續參數估測，因此不適合用分類交叉熵處理，而應使用回歸型損
失。SmoothL1Loss相較MSE更不易受少數極端誤差支配，適合用於此類多維係數回歸任務。

46def make_grid(n: int = 64):
47# Build a normalized coordinate grid over [-1, 1] x [-1, 1].
48x = np.linspace(-1.0, 1.0, n)
49y = np.linspace(-1.0, 1.0, n)
50xx, yy = np.meshgrid(x, y)
51rr = np.sqrt(xx**2 + yy**2)
52th = np.arctan2(yy, xx)
53pupil = rr <= 1.0
54return xx, yy, rr, th, pupil

57def zernike_radial(n: int, m: int, r: np.ndarray) -> np.ndarray:
58# Compute the radial part of a Zernike polynomial.
59m = abs(m)
60if (n - m) % 2 != 0:
61return np.zeros_like(r)
62out = np.zeros_like(r)
63for k in range((n - m) // 2 + 1):
64coeff = ((-1) ** k) * math.factorial(n - k)
65denom = (
66math.factorial(k)
67* math.factorial((n + m) // 2 - k)
68* math.factorial((n - m) // 2 - k)
69)
70out += coeff / denom * r ** (n - 2 * k)
71return out

74def zernike(n: int, m: int, r: np.ndarray, theta: np.ndarray) -> np.ndarray:
75# Combine radial and angular parts and normalize inside the pupil.
76rad = zernike_radial(n, m, r)
77if m > 0:
78z = rad * np.cos(m * theta)
79elif m < 0:
80z = rad * np.sin(abs(m) * theta)
81else:
82z = rad
83mask = r <= 1.0
84z = z * mask
85std = z[mask].std()
86if std > 1e-8:
87z = z / std
88return z.astype(np.float32)

105class FlatTopZernikeSimulator:
106"""
107Demo surrogate model.
108It starts from an ideal flat-top target image and uses Zernike coefficients
109to induce physically motivated warp/blur/shading distortions.
110
111In your real system, replace simulate() with either:
1121) your full Fourier-optics + SLM forward model, or
1132) real CCD measurements collected under known Zernike settings.
114"""
115def __init__(self, cfg: ForwardModelConfig):
116self.cfg = cfg
117self.xx, self.yy, self.rr, self.th, self.pupil = make_grid(cfg.image_size)
118self.target = self._make_target()
119self.basis = self._build_basis()
120# Precompute spatial gradients for later warp-field construction.
121self.basis_dx = np.stack([np.gradient(b, axis=1) for b in self.basis], axis=0)
122self.basis_dy = np.stack([np.gradient(b, axis=0) for b in self.basis], axis=0)
123
124def _make_target(self):
125# Build an ideal flat-top target using a high-order super-Gaussian.
126wx, wy, p = self.cfg.target_wx, self.cfg.target_wy, self.cfg.target_order
127tgt = np.exp(-((np.abs(self.xx) / wx) ** p + (np.abs(self.yy) / wy) ** p))
128return (tgt / tgt.max()).astype(np.float32)
129
130def _build_basis(self):
131return np.stack([zernike(n, m, self.rr, self.th) for n, m, _ in ZERNIKE_MODES], axis=0)
132
133def simulate(self, coeffs: np.ndarray, add_noise: bool = True):
134# Project the 15D coefficient vector onto the Zernike basis.
135coeffs = np.asarray(coeffs, dtype=np.float32)
136wavefront = np.tensordot(coeffs, self.basis, axes=(0, 0))
137dx = np.tensordot(coeffs, self.basis_dx, axes=(0, 0))
138dy = np.tensordot(coeffs, self.basis_dy, axes=(0, 0))
139
140# Use wavefront gradients to create spatial warping in the image.
141xw = self.xx + self.cfg.warp_scale * dx
142yw = self.yy + self.cfg.warp_scale * dy
143px = (xw + 1.0) * 0.5 * (self.cfg.image_size - 1)
144py = (yw + 1.0) * 0.5 * (self.cfg.image_size - 1)
145warped = map_coordinates(self.target, [py, px], order=1, mode=’constant’, cval=0.0)
146
147# Set blur strength from coefficient RMS to mimic defocus/high-order blur.
148blur_sigma = self.cfg.blur_scale * float(np.sqrt(np.mean(coeffs**2)))
149if blur_sigma > 1e-6:
150warped = gaussian_filter(warped, sigma=blur_sigma)
151
152# Use the Laplacian to create a local shading modulation term.
153shading = 1.0 + self.cfg.shading_scale * laplace(wavefront)
13
154shading = np.clip(shading, 0.75, 1.25)
155image = np.clip(warped * shading, 0.0, None)
156image = image / (image.max() + 1e-8)
157
158# Add sensor noise so the synthetic image better resembles CCD data.
159if add_noise:
160image = image + np.random.normal(0.0, self.cfg.noise_std, size=image.shape).astype(
np.float32)
161image = np.clip(image, 0.0, 1.0)
162return image.astype(np.float32)
163
164def flatness_metric(self, image: np.ndarray, roi_level: float = 0.8) -> float:
165# Compute relative standard deviation over the flat-top ROI as a flatness metric.
166mask = self.target > roi_level
167region = image[mask]
168return float(region.std() / (region.mean() + 1e-8))

228def run_epoch(model, loader, criterion, optimizer=None):
229training = optimizer is not None
230model.train(training)
231total_loss = 0.0
232total_mae = 0.0
233total_n = 0
234
235for x, y in loader:
236x = x.to(DEVICE)
237y = y.to(DEVICE)
238pred = model(x)
239loss = criterion(pred, y)
240if training:
241optimizer.zero_grad()
242loss.backward()
243optimizer.step()
244mae = (pred - y).abs().mean().item()
245bs = x.size(0)
246total_loss += loss.item() * bs
247total_mae += mae * bs
248total_n += bs
249return total_loss / total_n, total_mae / total_n

278def predict_coeffs(model: nn.Module, image: np.ndarray):
279# Run inference on one image and return the 15 predicted coefficients.
280model.eval()
281with torch.no_grad():
282x = torch.from_numpy(image[None, None, :, :]).float().to(DEVICE)
283pred = model(x).cpu().numpy()[0]
284return pred.astype(np.float32)

287def run_adaptive_loop(sim: FlatTopZernikeSimulator, model: nn.Module, true_coeffs: np.ndarray,
steps: int = 8, gain: float = 0.85):
288# Closed-loop correction: repeatedly estimate residual aberration and update compensation.
289correction = np.zeros_like(true_coeffs, dtype=np.float32)
290images = []
291residual_norms = []
292flatness = []
293predicted = []
294
295for _ in range(steps):
296residual = true_coeffs + correction
297image = sim.simulate(residual, add_noise=False)
298pred = predict_coeffs(model, image)
299correction = correction - gain * pred
300images.append(image)
301residual_norms.append(float(np.sqrt(np.mean(residual**2))))
302flatness.append(sim.flatness_metric(image))
303predicted.append(pred)
304
305final_residual = true_coeffs + correction
306final_image = sim.simulate(final_residual, add_noise=False)
307images.append(final_image)
308residual_norms.append(float(np.sqrt(np.mean(final_residual**2))))
309flatness.append(sim.flatness_metric(final_image))
310
311return {
312’images’: images,
313’residual_norms’: residual_norms,
314’flatness’: flatness,
315’predicted’: predicted,
316’final_residual’: final_residual,
317’final_correction’: correction,
318}
